##  Project Overview

This project aims to predict whether a customer will leave (churn) a company or not using Machine Learning classification algorithms.

Customer churn prediction helps businesses reduce customer loss by identifying customers who are likely to discontinue services.

This project demonstrates:
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Model training and evaluation
- Model comparison

---

##  Dataset Information

The dataset contains customer-related features such as:

- Customer ID
- Gender
- Senior Citizen
- Partner
- Dependents
- Tenure
- Phone Service
- Internet Service
- Contract Type
- Payment Method
- Monthly Charges
- Total Charges
- Churn (Target Variable: Yes/No)
